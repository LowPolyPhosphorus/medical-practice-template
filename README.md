## üõ†Ô∏è Customization Guide

### To Use This Template for Your Portfolio:

1. **Replace placeholder content:**
   - Practice name: "Summit Health"
   - Location: Gilbert, AZ
   - Phone: (480) 555-0123
   - Email: info@summithealth.com
   - Address: 123 Medical Plaza Dr

2. **Add real images:**
   - Replace `.team-photo` divs with actual provider photos
   - Replace `.hero-image-main` with professional images
   - Add service-specific images

3. **Create additional service pages:**
   - Copy `service-primary-care.html`
   - Rename and update content for each service
   - Update links in `services.html`

4. **Create provider profile pages:**
   - Copy `service-primary-care.html` as a template
   - Create `doctor-sarah-mitchell.html`, etc.
   - Add detailed bios, education, specialties

5. **Add Google Maps:**
   - Replace `.map-placeholder` in `contact.html`
   - Embed actual Google Maps iframe

6. **Set up forms:**
   - Connect forms to email service or CRM
   - Currently uses JavaScript alert (demo only)
